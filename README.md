# Fitness Website

A simple multi-page website built with **HTML, CSS, and JavaScript**.

This project is designed to showcase responsive layout, navigation between pages, and basic interactivity using vanilla JavaScript.

## Pages
- **Home** (`index.html`)
- **About** (`about.html`)
- **Training** (`training.html`)
- **Contact** (`contact.html`)

## Features
- Clean and responsive design
- Mobile-friendly layout
- Navigation menu
- Basic JavaScript interactivity

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Setup / How to Run
No server required. Simply open `index.html` in your browser:

```bash
# Open the project in your browser
open index.html  # Mac
start index.html # Windows
